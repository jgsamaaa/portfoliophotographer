## Tech Stack

**Client:** ReactJS, TailwindCSS, Framer-motion

## Run Locally

Clone the project

```bash
  git clone https://github.com/jgsamaaa/portfoliophotographer.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Installation
